# Engine!
A light-weighted Robot Simulator based on C++&GTK
- [x] Visualization
- [x] Start from scratch
- [x] With ros-like IPC method
- [x] Represent robots by URDF syntax 
- [x] Trajectory Generation 
- [x] Forward Kinematics
- [x] Forward Dynamics
- [x] Inverse Dynamics

# Install
```
mkdir build && cd build
cmake ../ && make -j8
./engine
```
